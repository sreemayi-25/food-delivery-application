# 🍽️ Craving Kitchen

**Craving Kitchen** is an online food ordering and delivery platform that connects customers with their favorite local restaurants. The platform provides a user-friendly interface for browsing menus, placing orders, and enjoying fast food delivery at your doorstep.

---

## 🔧 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  

---

## 🌟 Features

- 🛍️ Browse restaurant menus and dishes  
- 👤 User registration and login system  
- 🛒 Add items to cart and place orders  
- 📦 Order tracking and management  
- 📊 Admin panel for restaurant and order management  
- 📱 Responsive design for mobile and desktop
